# Tesla Executive Dashboard

Interactive financial dashboard analyzing Tesla revenue, profitability, market cap, energy growth, and stock performance.

---

## Features
- Revenue trend analysis
- Net income analysis
- Market capitalization
- Energy segment growth
- Stock price tracking
- Margin analysis

---

## Tech Stack
- Python
- Pandas
- Plotly
- Matplotlib
- Yahoo Finance API
